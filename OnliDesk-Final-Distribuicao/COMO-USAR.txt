🚀 COMO EXECUTAR O ONLIDESK
═══════════════════════════════════════════════════════════════════════════════

⚡ MÉTODO MAIS SIMPLES (RECOMENDADO):
1. Execute OnliDesk.bat (clique duplo)
2. O launcher verificará e instalará automaticamente o .NET se necessário
3. O OnliDesk será iniciado automaticamente

📋 ARQUIVOS INCLUÍDOS:
• OnliDesk.bat - Launcher inteligente (EXECUTE ESTE ARQUIVO)
• *.cs, *.xaml - Código fonte da aplicação
• OliAcessoRemoto.csproj - Arquivo de projeto
• Services/ - Serviços da aplicação
• Resources/ - Recursos e ícones
• COMO-USAR.txt - Este arquivo

🔧 REQUISITOS DO SISTEMA:
• Windows 10/11 (64-bit)
• .NET 8 Desktop Runtime (instalado automaticamente se necessário)
• Conexão com internet (para download do .NET se necessário)

✨ FUNCIONALIDADES DO LAUNCHER:
• ✅ Verificação automática do .NET 8 Desktop Runtime
• ✅ Download e instalação automática do .NET
• ✅ Execução via 'dotnet run' (mais confiável)
• ✅ Fallback para executável se necessário
• ✅ Interface amigável com instruções claras
• ✅ Tratamento robusto de erros

🌐 FUNCIONALIDADES DO ONLIDESK:
• ✅ Interface moderna e intuitiva
• ✅ Conexão segura com servidor OnliDesk (172.20.120.40:7070)
• ✅ Autenticação JWT
• ✅ Validação de IDs no formato XXX XXX XXX
• ✅ Sistema de configurações avançadas
• ✅ Gestão de conexões recentes
• ✅ Modo offline funcional
• ✅ Português brasileiro com acentuação correta

🆘 SOLUÇÃO DE PROBLEMAS:

❓ "OnliDesk não abre"
   → Execute OnliDesk.bat como administrador
   → Verifique se o antivírus não está bloqueando
   → Certifique-se de que todos os arquivos estão na mesma pasta

❓ "Erro de .NET"
   → Execute OnliDesk.bat novamente
   → Escolha instalação manual se automática falhar
   → Reinicie o computador após instalar .NET

❓ "Não conecta ao servidor"
   → Verifique conexão com internet
   → Confirme se o servidor 172.20.120.40:7070 está acessível
   → Use modo offline se necessário

🔗 LINKS ÚTEIS:
• Download .NET 8: https://dotnet.microsoft.com/download/dotnet/8.0
• Documentação .NET: https://docs.microsoft.com/dotnet/

💡 DICAS IMPORTANTES:
• SEMPRE execute OnliDesk.bat, não os arquivos .exe diretamente
• Mantenha todos os arquivos na mesma pasta
• Para desinstalar, simplesmente delete a pasta
• Crie um atalho do OnliDesk.bat na área de trabalho para facilitar o acesso

🎯 MÉTODO ALTERNATIVO (PARA DESENVOLVEDORES):
Se você tem o .NET 8 SDK instalado, pode executar diretamente:
1. Abra o prompt de comando nesta pasta
2. Execute: dotnet run
3. O OnliDesk será iniciado
