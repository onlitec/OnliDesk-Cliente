⚡ MÉTODO SUPER SIMPLES:
1. Execute OnliDesk.exe (clique duplo)
2. O programa verificará automaticamente se o .NET está instalado
3. Se não estiver, abrirá a página de download automaticamente
4. Após instalação do .NET, execute OnliDesk.exe novamente
5. O OnliDesk será iniciado automaticamente

📋 ARQUIVOS INCLUÍDOS:
• OnliDesk.exe - EXECUTÁVEL PRINCIPAL (EXECUTE ESTE ARQUIVO)
• OliAcessoRemoto.exe - Aplicação OnliDesk (usado internamente)
• Arquivos de suporte (.dll, .json) - Necessários para funcionamento
• COMO-USAR.txt - Este arquivo de instruções

🔧 REQUISITOS DO SISTEMA:
• Windows 10/11 (64-bit)
• .NET 8 Desktop Runtime (instalado automaticamente se necessário)
• Conexão com internet (para download do .NET se necessário)

🆘 SOLUÇÃO DE PROBLEMAS:

❓ "OnliDesk não abre"
   → Execute OnliDesk.exe como administrador
   → Verifique se o antivírus não está bloqueando
   → Certifique-se de que todos os arquivos estão na mesma pasta

❓ "Mensagem sobre .NET"
   → Clique em "Sim" para abrir a página de download
   → Baixe ".NET Desktop Runtime 8.0" para Windows x64
   → Execute o instalador baixado
   → Reinicie o OnliDesk.exe após a instalação

❓ "Não conecta ao servidor"
   → Verifique conexão com internet
   → Confirme se o servidor 172.20.120.40:7070 está acessível
   → Use modo offline se necessário

❓ "Antivírus bloqueia"
   → Adicione exceção para a pasta do OnliDesk
   → Marque OnliDesk.exe como confiável
   → Use Windows Defender se possível
